PHP na linha de comando:

1) php -a
2) comando echo "ola mundo";
3) cls para limpar terminal (cmd)


-------------------------------------------


-Para saber o tipo de uma variavel gettype(variavel)
- tipos de dados primitivos: integer, double, string e boolean
- Para conseguirmos concatenar uma string com variavel sem precisar
usar o ".", podemos usar as aspas duplas.
Ex: "meu nome é $nome";

Obs: Para o php interpretar caracteres especiais é importante
que ele esteja dentro de "".

-Para quebrar linha usamos o caracter especial \n (new line).
No windows usamos o retorno de carro, que é representado por \r\n.
Outra opção é utilizar o comando php, PHP_EOL.
ex: echo "Olá mundo!" . PHP_EOL;

-Para dar tab, o caracter especial é \t.
